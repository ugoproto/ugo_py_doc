Navigation¶

Bookmark, history.

bookmark -l

Current bookmarks:
name     -> path
untitled -> /home/user

Add a bookmark.

bookmark aaa path

bookmark -l

Current bookmarks:
aaa      -> path
name     -> path
untitled -> /home/user

Delete a bookmark.

bookmark -d aaa

bookmark -l

Current bookmarks:
name     -> path
untitled -> /home/user

Print the directory navigation history.

dhist

Directory history (kept in _dh)
0: /home/user
